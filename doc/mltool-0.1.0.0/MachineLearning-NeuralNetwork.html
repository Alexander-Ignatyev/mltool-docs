data Topology Source #

Neural network topology has at least 2 elements: numver of input and number of outputs. And sizes of hidden layers between 2 elements. Bias input must not be included.

makeTopology :: Int -> Int -> [Int] -> Topology Source #

Creates toplogy. Takes number of inputs, number of outputs and list of hidden layers.

initializeTheta :: Topology -> IO Vector Source #

Create and initialize weights vector with random values for given neural network topology.

predictMulti :: Topology -> Matrix -> Vector -> Vector Source #

Calculate output for given input matrix (X) and list of weight vector (theta). Returns vector of predicted class indices (assuming class indices starts at 0).

calcAccuracy :: Vector -> Vector -> R Source #

Calculates accuracy of Classification predictions. Takes vector expected y and vector predicted y. Returns number from 0 to 1, the closer to 1 the better accuracy. Suitable for both Classification Types: Binary and Multiclass.
